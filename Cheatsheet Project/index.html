CSS Display & Positioning Cheatsheet

Property Value(s) Description
z-index +/- integer specifies how far back or how far forward an element will appear on a web page when it overlaps other elements.
display block, inline, inline-block Block-level elements take up the full width of their container with line breaks before and after, and can have their height and width manually adjusted. Inline elements take up as little space as possible, flow horizontally, and cannot have their width or height manually adjusted. Inline-block elements can appear next to each other, and can have their width and height manually adjusted.
position static, fixed, absolute, relative When the CSS position has a value of fixed, it is set/pinned to a specific spot on a page. The fixed element stays the same regardless of scrolling. The value absolute for the CSS property position enables an element to ignore sibling elements and instead be positioned relative to its closest parent element that is positioned with relative or absolute. The value relative of the CSS position property enables an element to be positioned relative to where it would have originally been on a web page.
clear none, left, right, both The CSS clear property specifies how an element should behave when it bumps into another element within the same containing element. The clear is usually used in combination with elements having the CSS float property. This determines on which sides floating elements are allowed to float.
float left, right The CSS float property determines how far left or how far right an element should float within its parent element.
